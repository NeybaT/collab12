# Artificial intelligence-enhanced FTIR spectroscopy for simultaneous multicomponent carbohydrate quantification in dairy matrices

## Description

This repository contains the data and code used for the development of an AI-based FTIR framework for carbohydrate quantification in dairy samples.

The study integrates Fourier Transform Infrared (FTIR) spectroscopy with chemometric and machine learning models, including Partial Least Squares (PLS) regression and Artificial Neural Networks (ANN), to predict lactose, glucose, and galactose concentrations.
